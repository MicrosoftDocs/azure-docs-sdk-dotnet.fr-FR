# <a name="azure-mobile-libraries-for-net"></a>Bibliothèques Azure Mobile pour .NET

## <a name="client-library"></a>Bibliothèque de client

Installez le [package NuGet](https://www.nuget.org/packages/Microsoft.Azure.Mobile.Server) directement à partir de la [Console du Gestionnaire de package][PackageManager] Visual Studio ou avec la [CLI .NET Core][DotNetCLI].

#### <a name="visual-studio-package-manager"></a>Gestionnaire de package Visual Studio

```powershell
Install-Package Microsoft.Azure.Mobile.Server
```

```bash
dotnet add package Microsoft.Azure.Mobile.Server
```
